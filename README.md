# KUKA HOME Landing

Premium furniture landing page built with semantic HTML, CSS and vanilla JavaScript.

## Structure

- `index.html` - main landing page
- `styles/main.css` - full responsive styling and animations
- `scripts/data.js` - brand constants, demo content and translations
- `scripts/app.js` - reusable render functions and interactions
- `assets/logo-mark.svg` - brand mark placeholder

## Run

Open `index.html` directly in a browser, or run a simple local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
